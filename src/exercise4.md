# Эндпоинты

## CREATE
Добавить рецепт  
HTTP-метод - POST:  
__Эндпоинт: POST /api/v1/recepts__

## READ
Просмотреть рецепты  
HTTP-метод - GET:

__Эндпоинт: GET /api/v1/recepts__

## UPDATE
Редактировать рецепт   
HTTP-метод - PUT:

__Эндпоинт:PUT /api/v1/recepts/1__

## DELETE
Удалить рецепт   
HTTP-метод - DELETE:

__Эндпоинт: DELETE /api/v1/recepts/1__
